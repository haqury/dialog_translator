## [1.0.0] - YYYY-MM-DD

### Добавлено
- Распознавание речи через Google Web Speech API
- Автоматический перевод через Google Translate API
- Озвучивание переводов через ElevenLabs TTS
- Поддержка 5 языков: русский, английский, испанский, французский, немецкий
- GUI на PyQt5 с настраиваемым интерфейсом
- Сохранение настроек между запусками
- Автоопределение языка говорящего
- Ручной ввод текста
- Настройки микрофона и динамиков
- История сообщений с возможностью очистки
- Полноэкранный режим
